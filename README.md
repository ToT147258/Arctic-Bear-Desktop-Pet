# 北极熊桌面宠物系统

这是一个全新的桌面宠物项目，计划基于 Python + PySide6 实现。

项目目标是参考旧桌宠项目的功能结构，重新设计并实现一个以北极熊为形象的桌面宠物系统。旧项目只作为功能参考和软件工程分析依据，新项目不使用 `run_DyberPet.py` 作为入口。

## 项目定位

- 主框架：PySide6 桌面应用
- 新项目入口：`main.py`
- 最终形象：北极熊桌宠
- 当前阶段：中期答辩原型与软件工程设计
- 当前目标：先完成可运行的桌面应用原型，再逐步替换真实北极熊素材和动作系统

## 五大模块

1. 桌宠形象与桌面交互模块
2. 宠物状态与成长任务模块
3. 背包、商店与喂养模块
4. 系统设置与存档管理模块
5. 通知日志与创新展示模块

## 运行方式

```bash
pip install -r requirements.txt
python main.py
```

运行后会出现两个窗口：

- 北极熊桌宠控制台：用于展示五大模块。
- 北极熊悬浮桌宠窗口：使用半扁平、生动的北极熊 PNG 素材，用于中期演示桌宠形象、置顶显示、呼吸动画和拖拽交互。
