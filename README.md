# Arctic Bear Desktop Pet

一个基于 Python + PySide6 的北极熊桌面宠物项目。项目包含悬浮桌宠窗口、真实序列帧动作、主控台、状态成长、背包投喂、任务奖励、设置存档和通知日志，适合作为课程设计、中期答辩或个人桌宠项目继续扩展。

## 当前效果

- 真实北极熊 PNG 序列帧动作：待机眨眼微动、走路、跳跃、挥手、互动、拖拽、睡觉
- 悬浮桌宠窗口：无边框、置顶、透明背景、可拖拽、右键菜单、缩放比例
- 主控台：总览工作台 + 五大功能模块
- 状态成长：饱食度、心情、体力、好感度、等级、经验、金币
- 背包商店：投喂、购买、使用物品，自动影响状态
- 每日任务：登录、陪伴、投喂、互动、睡觉、走路任务
- 设置存档：缩放、透明度、置顶、走路移动开关、本地 JSON 存档
- 通知日志：记录操作行为，并可在桌宠旁显示气泡提示

## 运行方式

建议使用 Python 3.10 或更新版本。

```bash
pip install -r requirements.txt
python main.py
```

启动后会出现两个窗口：

- `北极熊桌宠控制台`：用于查看和操作五大模块。
- `北极熊悬浮桌宠`：显示真实动作帧，可拖拽、右键切换动作、Ctrl + 鼠标滚轮缩放。

## 交互方式

- 单击桌宠：触发互动动作
- 双击桌宠：触发挥手动作
- 按住拖动：移动桌宠，并播放拖拽动作
- 右键桌宠：打开动作和缩放菜单
- 控制台按钮：可触发走路、跳跃、睡觉、投喂、任务、设置等行为

## 项目结构

```text
main.py                         # 应用入口
src/app.py                      # 主控台窗口
src/pet_window.py               # 悬浮桌宠动作窗口
src/pet_data.py                 # 状态、背包、任务、日志、存档数据中心
src/modules/                    # 五大功能模块页面
assets/polar_bear/role/PolarBear # 北极熊动作配置与 PNG 序列帧
docs/                           # 项目文档、模块说明、UML 材料
tools/                          # 动作帧处理和导入工具
```

## 存档说明

运行时会自动生成：

```text
data/save.json
```

这个文件保存用户状态、背包、任务和设置，已经被 `.gitignore` 忽略，不会上传到 GitHub。设置页可以导出 `data/save-export.json`，也可以从这个文件导入。
