# 背包、商店与喂养模块

## 模块作用

该模块负责物品管理、商店购买和桌宠喂养功能。它与宠物状态模块密切相关，用户通过投喂食物或使用道具，可以影响北极熊桌宠的饱食度、心情值和好感度。

## 主要功能

- 背包物品展示
- 食物道具管理
- 商店购买
- 投喂操作
- 物品数量变化
- 投喂后状态变化
- 金币或奖励机制
