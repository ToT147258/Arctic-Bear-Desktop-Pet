## 三、实验小细节

​	1.当你使用云算力的时候，你会发现，当你本地关机的时候，实验就断了，那你想要他挂机怎么办呢

​		**当然是使用Screen啦**

```python
#检查是否已安装
which screen

#如果没有安装，通过如下命令进行安装
sudo apt-get install -y screen

#最后那个为命名，创建会话
screen -S rece

#获得目录
screen -ls

#进入会话，最后那个是名称
screen -r 3290.rece

#显示 (Attached) 说明上次的连接还没断开。强制接管它：
screen -d -r 81744.RECE

#开启日志，-Logfile ~/mylog.txt：指定日志保存路径（这里保存在家目录下的 mylog.txt 文件中）。若不写此参数，默认保存为当前目录下的 screenlog.0。（screen是只有一小个窗口的，建议用个文件记下日志）
mkdir -p ~/screenlogs
screen -L -Logfile ~/screenlogs/attack.log -S attack

#看日志：
tail -f ~/screenlogs/attack.log
#ctrl+c退出日志

#杀死进程：
screen -S 名字 -X quit
```
