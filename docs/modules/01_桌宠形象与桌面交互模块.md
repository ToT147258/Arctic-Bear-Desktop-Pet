# 桌宠形象与桌面交互模块

## 模块作用

该模块是北极熊桌面宠物系统的核心展示模块，主要负责桌宠在桌面上的显示、移动和基础互动。用户打开应用后，首先看到的就是悬浮在桌面上的北极熊桌宠，因此该模块直接影响系统的第一印象和交互体验。

## 主要功能

- 北极熊桌宠形象显示
- 桌宠窗口置顶显示
- 鼠标拖拽移动
- 双击互动反馈
- 呼吸动画
- 后续扩展待机、点击、睡眠、投喂等动作

## 中期已完成
