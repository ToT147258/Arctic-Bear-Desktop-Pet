# 北极熊桌面宠物系统中期答辩方案

## 一、项目说明

本项目是一个全新的桌面宠物项目，项目名称为“北极熊桌面宠物系统”。项目计划基于 Python + PySide6 开发，最终实现一个以北极熊为形象的桌面宠物应用。

旧项目只作为参考对象，用来分析桌面宠物系统应包含哪些功能，例如桌宠展示、拖拽互动、状态养成、背包喂养、任务成长、系统设置和通知日志等。新项目不是旧项目的简单运行或复制，也不使用旧项目的 `run_DyberPet.py` 作为入口。

当前新项目入口为：

```text

```

## 七、桌面应用原型说明

当前新项目是纯 PySide6 桌面应用。运行 `python main.py` 后，应用包含两个部分：

- 北极熊桌宠控制台：展示五大模块和中期功能规划；
- 北极熊悬浮桌宠窗口：使用半扁平、生动的北极熊 PNG 素材，用于展示桌宠形象、置顶显示、呼吸动画和鼠标拖拽交互。

中期答辩时可以说明：当前应用已经形成桌面端原型，后续会在真实北极熊素材基础上继续补充序列帧、骨骼动画或更多动作资源。
